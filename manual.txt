Q 환급결정 순서
보통 소득세과에서 환급결정 후, 징세과로 통보가 넘어가고 징세과에서 최종 체납 충당 및 환급처리 합니다.

경정청구 환급 처리 절차 프로세스
경정청구 접수 → 경정청구 검토 → 경정청구 환급결정 → 소득세과장 결재 → 징세과 통보전달
→ 징세과 통보확인 → 통보내용 검토 → 체납/농특세 충당처리 → 징세과장 결재 → 지급 결정

- 소득세과 담당조사관 처리 구간: 경정청구 접수 ~ 징세과 통보전달 (약 2개월 소요)
- 징세과 환급 담당자 처리 구간: 징세과 통보확인 ~ 지급 결정 (약 1주일 소요)

부서별 업무

소득세과
1. 접수된 경정청구 접수
2. 경정청구서 검토
3. 경정청구 최종 환급결정
>경정감 처리는 담당 조사관이 합니다. 

징세과
1. 소득세과에서 넘어온 환급통보 확인
2. 통보된 내용 검토
3. 체납 / 농특세(or 국세) 충당처리
4. 환급금 지급결정

참고사항
- 체납 충당처리는 징세과에서 처리하고 있기에, 소득세과보다 징세과에서 정확한 확인이 가능함.
  소득세과에서 충당 여부 확인이 불가하면 징세과로 최종 확인 요청.
- 계좌 등록은 지급 결정하고 있는 징세과에서 처리하고 있으니, 소득세과보다는 징세과로 문의.

Q 농특세가 무엇인가요?
-농어촌특별세: 농어촌의 개발과 지원을 위해 특정 세금에 추가로 부과되는 목적세 공제세액의 20% 부과
-지수에서의 신고 항목: 고용증대, 통합고용

Q 농특세 환급액은 뭔가요?
-고용이 증가 시, 증가한 인원 수만큼 세금이 줄어들고 그에 따른 환급도 받을 수 있습니다. 이때 감면받은 세액의 20%는 농어촌특별세로 납부해야 합니다.
-고용 증가 이후에 고용이 다시 감소하면 이전에 감면받았던 세금을 다시 납부해야 하며, 해당 감면 혜택은 취소된 것으로 간주되어 이에 따라 기존에 납부한 농어촌특별세는 환급받게 됩니다.
-담당 조사관에게는 "수정신고 시 환급세액과 공제새액이 감소하고, 공제세액이 감소하면서 농특세도 감소되기에 차액인 농특세 환급액이 발생하게 됩니다."로 대응하면 돼요.

Q 조사관이 처리기한 지났는데 취하 후 재접수 하라고 합니다. 어떻게 대응 해야 할까요?
-조사관에게 "처리기한 지났는데 저희쪽에서 취하해야 할 사유가 있을까요? 기한내로 처리 어려우시면 기각처리 해주세요" 로 대응 해주세요.  통화날에 기각 워크플로 작성하지 말고, 다음 인용확인 했을 때 기각 사실 확인되면 워크플로 작성해주시면 됩니다.

-결정확인필요(B-3): 홈택스 로그인 이후, 홈택스 결정내역 확인되는 상황 (인용확인 대상 아닙니다.)
-환급결정(B-4): 인용확인(환급결정) 이후 정상 결정된 상황 (인용확인 대상 아닙니다.)

Q 1만원 미만은 고지 안 된대요 
- 국세기본법 시행령 제65조의 3 (고지금액의 최저한도)에 따라 고지할 수 있는 금액(대통령령으로 정하는 금액)은 최소 1만원으로 지정되어 있습니다.
-1만원 미만의 농특세가 충당되지 않은 경우엔  특이사항에만 잘 기입해주시고 바로 환급결정으로 단계변경 해주셔도 되고, 워크플로 작성하지 않아도 됩니다. 
-특이사항으로 충당되는 경우도 있었다고 하니, 정상 충당일 경우 평소처럼 단계변경 해주시면 됩니다.

Q 경정청구 신고서와 위임장에 찍힌 건 막도장 아닌가요?
-조사관에게 "납세자와 전자계약(카카오 인증)을 하고 있습니다. 전자계약을 통해 생성된 위임장의 서명 또는 도장은 이미지 파일 일 수 밖에 없습니다" 라고 대응해주세요. 관련하여 전자서명법령도 있는데 설명드릴지 확인 후에 전자서명법 제 3조에 대해 설명하면 됩니다. 
-전자서명법 제3조(전자서명의 효력)
① 전자서명은 전자적 형태라는 이유만으로 서명, 서명날인 또는 기명날인으로서의 효력이 부인되지 아니한다.
② 법령의 규정 또는 당사자 간의 약정에 따라 서명, 서명날인 또는 기명날인의 방식으로 전자서명을 선택한 경우 그 전자서명은 서명, 서명날인 또는 기명날인으로서의 효력을 가진다.

Q 부양가족 중복공제, 소득요건 미충족으로 환급액 변동되었어요
-부양가족 인적공제(=소득공제)는 요건에 해당 되는 경우에만 받는데요
소득공제를 받으면 내야할 세금을 줄이는 것이라 생각 해주시면 됩니다.
-부양가족 공제를 받기 위해서는 요건들이 모두 충족 되어야만 합니다. 요건 미충족으로 확인되면 공제 받았던 부양가족을 배제 해야합니다.
- 부양가족 배제 시, 이전에 환급 or 추가로 공제 받은 금액에 대해서는 고지될 수 있습니다. (과소신고로 고지 되는 것.)
-결론: 조사관이 요건 미충족 부양가족 제외해보니 환급 나올것이 없다고 하면 납부고지는 따로 안 하는건지 확인해주시면 됩니다.
-부양가족 누가, 왜 제외 되었는지 상세 확인이 필요합니다. 부양가족 확인 방법은 아래와 같습니다.
1. 분석상세2 접속 
2. [신고서 정보] 클릭
3. 신고서식 - 변동된 연도 클릭
4. 신고연도 - 종합소득세 - 신고서         모양 클릭
5. (11) 소득공제명세서 찾기
6. 인적공제 대상자 명세에서 누가, 왜, 어떠한 공제가 제외 되었는지 확인
-조사관에게 "몇년도 귀속, 부양가족 누가 제외된걸까요?" 라고 물어봐주세요. 만약 기본공제와 추가공제가 함께 있다면 기본공제 외에 추가공제도 같이 받았던건지 물어봐주세요 
-확인요청_인용확인 (환급액 변동)으로 워크플로 작성 해주시고, 인용확인상태 [대응필요] 기입하면 됩니다.
-이 외의 필드값 기입과 단계변경은 금지입니다. 


Q. 중소기업 청년 취업자 소득세 감면 (중취감)
-저희는 현재 중특감만 신고하고 있고, 중취감은 신고하지 않습니다.
-조사관이 중특감을 중취감으로 보고 잘못 기각했다면 중특감과 중취감이 다른 내용인 점 전달하고 재 검토 요청 해주시면 됩니다.

Q 환급액이 음수(-)/원복으로 기재되어 있어요 
-환급액 중에서 [원복] 혹은 캡쳐본처럼 받아야 할 환급액보다 납부가 더 크거나(음수), 납부만 있는 경우엔 관리자에게 우선 공유 부탁드립니다.
-[환급결정]으로 절대 단계변경 하시면 안 됩니다.

Q. 환급계좌 개설 
-환급계좌 개설 필요한 경우 [환급결정]으로 직접 단계변경 하지 않으셔도 됩니다.
1.담당/징세과로 계좌등록 요청 거부 시
2.확인요청_인용확인 워크플로 작성
파드에서의 환급계좌 일치여부- 현금수령, 입금예정일- 공란 해두시면 돼요.
3. 관리자가 고객지원팀으로 홈택스 환급계좌 개설 요청
4.고객이 직접 홈택스 환급계좌 개설 완료
5.관리자가 [환급결정]으로 단계변경

※ 환급계좌 개설은 담당 조사관이 아닌 징세과로 우선 부탁드리며, 연도별로 500만원 초과건은 세무서에서 환급계좌 등록 불가합니다.
※ 이미 현금수령 한 것이라면 상관 없습니다! 환급계좌 일치여부 [현금수령]으로 기입, 입금 에정일은 수령일로 기압하시고 단계변경 하셔도 돼요.
